# 一个用C++实现的DFT算法
## 文件
* DSP_DFT: C++源代码。
* DFTplot: Matlab源代码用于画出图像。
* input: 变换前的信号数据，用txt储存。
* output: 变换后的信号数据，用txt储存。
## 说明
运行c++文件后会重写input和output文件，用matlab读取input和output文件只管显示。Matlab中既画出了用该算法实现的波形，又画出了用Matlab中函数画出的波形，可对比观察
